# RetoCNTwinforms
El proyecto fue desarrollado en .Net Framework 4.6

Para llevar a cabo la ejecución del mismo se debe descargar el proyecto, abrir la solución, dirigirse a la clase llamada basededatos.cs, modificar el método llamado ObtenerConexion() y allí cambiar el parametro Server, en el debera especificar el nombre o dirección ip del servidor donde tenga instalado sql server y que a su vez le permita el ingreso al mismo, Acto seguido deberá ejecutar el script .sql llamado CntNetCore.sql desde su gestor de base de datos SQL Server.

Una vez creada la base de datos por medio del script .sql anteriormente ejecutado, se debe regresar a la solución abierta desde visual studio y allí oprimir la tecla f5 para iniciar la ejecucición del mismo.
